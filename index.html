Sources and Notes

Primary series: CES0000000001, all employees, thousands, total nonfarm, seasonally adjusted.

Method: signed monthly change = current-month BLS total nonfarm payroll employment level minus prior-month level. Values are shown in thousands of jobs.

Important: archived Employment Situation releases can later be revised. This dashboard uses the data in the workbook supplied for the GitHub Pages site.
